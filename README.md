<h2 id="usage" > 👷 Usage </h2>

Required! Install git, node and yarn (or npm).

```bash
# Clone Repository
$ git clone https://github.com/pedroribeiroo/oak.git

# Install Dependencies
$ yarn

# Run Server Aplication
$ yarn server

# Run Aplication
$ yarn dev

# Access localhost
```
